## 📝 What to Expect

1. **Enter a Topic** when prompted (e.g., "Artificial Intelligence")

2. **Browser will open automatically**
   - Chrome will navigate to gemini.google.com
   - If not logged in, log in manually
   - Press Enter in the terminal to continue

3. **Wait 4-6 minutes** while the system:
   - Researches your topic with Gemini
   - Generates 20-30 slides
   - Creates professional visualizations
   - Assembles the PowerPoint

4. **Your presentation is ready!**
   - Saved in `output/[Topic]_[Timestamp]/`
   - Folder opens automatically
   - Open the .pptx file in PowerPoint

## 🛠️ Troubleshooting

### Browser doesn't open?
- Make sure Chrome is installed
- Check internet connection
- Run as administrator if needed

### Stuck at "Waiting for Gemini"?
- Check if you're logged into Google
- Verify internet connection
- Browser might need manual interaction

### No visualizations generated?
- Don't worry! Sample visualizations are created as fallback
- Check the `charts/` folder in your output

### Want to change colors or theme?
- Edit `modules/ppt_builder.py` for PowerPoint colors
- Edit `modules/visualizations.py` for chart colors
